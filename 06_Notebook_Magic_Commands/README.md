# Lab 6: Notebook Magic Commands

This lab introduces Databricks Notebook Magic Commands, allowing multi-language code execution and utilities.

## Step-by-Step Guide

### Step 1: Access Workspace and Open Notebook
- Navigate to Databricks workspace home page.
- Open an existing notebook.

![Access Workspace and Open Notebook](https://github.com/saliksalik/DATA-BRICKS/blob/main/06_Notebook_Magic_Commands/outputs/Access%20Workspace%20and%20Open%20Notebook.png?raw=true)

### Step 2: Check and Change Default Language
- View default language (e.g., Python) at the top.
- Click to change if needed; mix languages with magic commands.

![Check and Change Default Language](https://github.com/saliksalik/DATA-BRICKS/blob/main/06_Notebook_Magic_Commands/outputs/Check%20and%20Change%20Default%20Language.png?raw=true)

### Step 3: Use %sql for SQL Code
- Type %sql in a cell to run SQL.
- Write and execute SQL expressions.

![Use %sql for SQL Code](https://github.com/saliksalik/DATA-BRICKS/blob/main/06_Notebook_Magic_Commands/outputs/Use%20%25sql%20for%20SQL%20Code.png?raw=true)

### Step 4: Use %scala and %python for Other Languages
- %scala for Scala code; %python for Python (optional in Python notebooks).

![Use %scala and %python for Other Languages](https://github.com/saliksalik/DATA-BRICKS/blob/main/06_Notebook_Magic_Commands/outputs/Use%20%25scala%20and%20%25python%20for%20Other%20Languages.png?raw=true)

### Step 5: Use %md for Markdown Documentation
- Type %md to create notes with Markdown formatting.

![Use %md for Markdown Documentation](https://github.com/saliksalik/DATA-BRICKS/blob/main/06_Notebook_Magic_Commands/outputs/Use%20%25md%20for%20Markdown%20Documentation.png?raw=true)

### Step 6: Use %fs for File System Commands
- Type %fs to run DBFS commands (e.g., ls for listing files/datasets).

![Use %fs for File System Commands](https://github.com/saliksalik/DATA-BRICKS/blob/main/06_Notebook_Magic_Commands/outputs/Use%20%25fs%20for%20File%20System%20Commands.png?raw=true)

### Step 7: Use %sh for Shell Commands and Explore Magics
- %sh for Linux commands; %lsmagic to list all commands; ? for help.

![Use %sh for Shell Commands and Explore Magics](https://github.com/saliksalik/DATA-BRICKS/blob/main/06_Notebook_Magic_Commands/outputs/Use%20%25sh%20for%20Shell%20Commands%20and%20Explore%20Magics.png?raw=true)

### Step 8: Run Notebook and Manage Execution
- Click "Run All" to execute entire notebook; manage cells and state.
