# Lab 2: Introduction to Databricks Workspace

This lab provides a walkthrough of the Databricks Workspace UI, explaining key menus and functionalities.

## 1) Homepage Overview
- Access the Databricks workspace homepage.
- View shortcut links at the top for day-to-day activities.
- Note recent items and favorites (may be empty initially).

![Screenshot 1](outputs/Homepage Overview.png)

## 2) Left Side Menu - Workspace
- Expand the left side menu (hover to open if collapsed).
- Click Workspace menu to access code, directories, files, and folders.
- View Home, Shared folder, and Users folder.
- Navigate to your home directory (e.g., /Users/[username]).

![Screenshot 2](outputs/Left Side Menu - Workspace.png)

## 3) Repos in Workspace
- In Workspace menu, see Repos option.
- Connect workspace to source control like GitHub or Azure DevOps.
- Files saved in source control can be accessed and committed from here.

![Screenshot 3](outputs/Repos in Workspace.png)

## 4) Other Menus: Recent, Catalog, Workflow, Compute
- Recent: List of recent items.
- Catalog: Catalog Explorer for metadata (databases, tables, views).
- Workflow: Create and manage workflows, jobs, and dependencies.
- Compute: Create and manage clusters (all-purpose, job compute, SQL warehouses).

![Screenshot 4](outputs/Other Menus.png)

## 5) Data Ingestion and Delta Live Tables
- Data Ingestion: Use partner solutions or native integrations to ingest data.
- Delta Live Tables: Declarative ETL offering, accessible via Workflow menu.

![Screenshot 5](outputs/Data Ingestion and Delta Live Tables.png)

## 6) SQL Menu
- Databricks offers serverless data warehouse capabilities.
- Create SQL warehouses, define queries, use SQL editor, create dashboards and alerts.

![Screenshot 6](outputs/SQL Menu.png)

## 7) Machine Learning Menu
- Platform for implementing ML algorithms.
- Access ML functionalities under this menu.

![Screenshot 7](outputs/Machine Learning Menu.png)

## 8) Marketplace and Partner Connect
- Marketplace: Powered by Delta Sharing, access data from vendors (free/paid).
- Partner Connect: Integrate with partners for data ingestion, visualization, preparation, ML, security, governance.

![Screenshot 8](outputs/Marketplace and Partner Connect.png)

## 9) Top Right Menu
- Expand menu at top right.
- Access User Settings, Admin Settings, Manage Account, and logout.
- Links to Azure portal and Databricks admin console.

![Screenshot 9](outputs/Top Right Menu.png)
